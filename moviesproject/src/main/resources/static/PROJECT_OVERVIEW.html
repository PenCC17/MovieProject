High-level summary

This is a Spring Boot (Java 17) REST API for a movies/reviews system. It uses Spring Web, Spring Data JPA, H2/MySQL connectors and Lombok. Main responsibilities:

  • Account management (register/login)
  • Movie CRUD basics (list, add, lookup)
  • Review submission, listing and editing

Entities

Domain model using JPA annotations and Lombok for boilerplate.

Account

@Entity @Table(name="account")
public class Account {
    @NotBlank @Size(min=3,max=20) String username;
    @NotBlank @Size(min=6,max=40) String password;
    @Id @GeneratedValue(strategy = GenerationType.AUTO) private Long accountId;
}

Movie

@Entity @Table(name="movie")
public class Movie {
    @NotBlank String name;
    @NotBlank String director;
    Long releaseYear;
    @Id @GeneratedValue(strategy=GenerationType.AUTO) private Long movieId;
}

Review

@Entity @Table(name="review")
public class Review {
    @ManyToOne @JoinColumn(name="account_id") Account accountId;
    @ManyToOne @JoinColumn(name="movie_id") Movie movieId;
    @Size(max=255) String reviewText;
    Integer rating;
    @Id @GeneratedValue(strategy = GenerationType.AUTO) private Long reviewId;
}
